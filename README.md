# Single-Page Resume Website (HTML Only)

A clean, single-page resume 
It includes semantic sections like **Header/Name, About, Skills, Experience Table, Projects, Education Table, and Contact Details**.

---

## Features Included
- Semantic structure using: `header`, `main`, `section`, `article`, `footer`
- Skills list using `<ul>`
- Experience section using a proper `<table>` with `<thead>` and `<tbody>`
- Projects section with multiple project entries
- Education section with a table
- Contact details section with links (email, LinkedIn, GitHub, etc.)

## Setup (Run Locally)
You can run this project locally in **any browser**. 

### Option 1: Download
1. Download the index.html file to your local machine.
2. double-click on the downloaded `index.html` file.
4. Your browser will open the webpage.


### Option 2: Using VS Code Live Server (Recommended)
1. Install **VS Code**
2. Install the extension: **Live Server**
